# csvDataCleaningTool

A collection of practical Python automation tools for real-world data processing tasks.

This repository focuses on small, reliable workflows commonly required in data cleaning, validation, and reporting jobs.

---

## Included tools

### 1. CSV Cleaner
Automates the process of cleaning and validating CSV datasets.

**Features:**
- Validates required columns
- Safely skips invalid or incomplete rows
- Generates clean output CSV files
- Provides detailed execution logs
- Configurable via YAML (no hardcoding)

Use cases include data cleanup, analytics preparation, and CSV exports from business systems.

---

### 2. CSV Summary Report Generator
Generates clear, human-readable summary reports from CSV data.

**Features:**
- Counts total, valid, and skipped rows
- Produces readable summary reports
- Provides email-domain breakdowns
- Designed for non-technical stakeholders
- Safe to re-run with timestamped outputs

Useful for data audits, validation reports, and dataset overviews.

---

## Design principles

- Simple, reusable automation scripts
- Configuration-driven behavior
- Clear logging and outputs
- Safe and predictable execution
- Built for small automation and freelancing use cases
